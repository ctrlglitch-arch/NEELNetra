# 🌊 NeelNetra — River Intelligence System
### *"Turning Water Data into Actionable Intelligence"*

---

## 📌 Project Overview
NeelNetra is a data-driven web platform designed to analyze and interpret river pollution data in a simple and meaningful way.

It allows users to input the name of a river and instantly receive insights such as:
- 📊 Pollution Index  
- 🧪 Dominant Contaminants  
- ⚠️ Potential Health Risks  
- 📈 Historical Trends  

The system bridges the gap between raw environmental data and real-world understanding by converting complex chemical parameters into user-friendly insights.

---

## ❗ Problem Statement
Rivers are a vital natural resource, but they are increasingly polluted due to:
- Industrial waste  
- Sewage discharge  
- Poor monitoring systems  

### Key Challenges:
- Environmental data is scattered and not easily accessible  
- Chemical parameters like **BOD, DO, and pH** are difficult for common people to understand  
- No simple platform exists to evaluate river health and associated risks  

---

## ✅ Our Solution
NeelNetra transforms raw river data into clear, actionable insights:

- 📊 Calculates a **Pollution Index**  
- 🧪 Identifies the **Dominant Pollutant**  
- ⚠️ Predicts **Health Risks**  
- 📈 Visualizes **10-year pollution trends**  
- 🌐 Provides a clean and interactive UI  

👉 Making environmental data accessible to everyone — not just experts.

---

## ⚙️ How It Works
1. User enters a river name  
2. Backend fetches historical data  
3. Pollution index is calculated  
4. Dominant pollutant is identified  
5. Health risks are evaluated  
6. Trend graph is generated  
7. Results are displayed on the frontend  

---

## ⚙️ Tech Stack
- **Backend:** FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **Visualization:** Matplotlib
